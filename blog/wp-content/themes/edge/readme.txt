== Installation ==
1. Primary: Login to your wp-admin area and go to Dashboard -> Appearance -> Themes. Select Install tab and click on Upload link. Select theme .zip and click on Install now button. If you have any errors, use can also use an alternate method.

2. Alternate: Unzip the template file (edge.zip) that you have downloaded. Via FTP, upload the whole folder (edge) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(edge). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/edge/index.php (for example).

3. Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Edge theme to activate it.

4. Want to enjoy Edge feature. Go to  Dashboard -> Appearance -> Customize and play with Edge Options

5. The theme is offered free of charge. If someone asked money for it, someone just tricked you.
